# Incident Report

## Alert Summary
Suspicious email activity detected from external IP 185.23.45.2.

## Investigation
Email logs were analysed to identify phishing patterns and user interaction with malicious links.

## Findings
- External sender delivered phishing emails
- Multiple users clicked the malicious link
- Same domain used across all emails

## Impact
Potential credential harvesting and account compromise.

## Recommendation
- Block the malicious domain
- Reset affected user credentials
- Conduct user awareness training
