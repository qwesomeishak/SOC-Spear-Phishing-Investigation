# Spear Phishing Investigation Using Splunk

## Project Overview
This project demonstrates how a SOC analyst can investigate a spear phishing attack using Splunk.

The investigation focuses on identifying malicious email senders, phishing links, and user interaction with suspicious emails.

## Scenario
Multiple users received emails from an external sender containing a suspicious link. Several users clicked the link, indicating a potential phishing attack.

## Data Source
The dataset used in this project was simulated to represent realistic email security logs for educational purposes.

## Key Findings
- Suspicious external sender identified
- Malicious phishing link detected
- Multiple users clicked the phishing link
- Potential credential harvesting risk

## Skills Demonstrated
- Email threat investigation
- Phishing detection
- Threat intelligence analysis
- SPL query development
- SOC incident reporting
